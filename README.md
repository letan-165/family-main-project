# Bun Nuoc Co Le - DevWeb

## Introduction
A web application built to manage my family's small food business operations.

This project is also used as a personal portfolio to demonstrate my backend development skills.

## Project Goal
The goal of this project is to design and implement a system with a clean and maintainable architecture.

Key focuses include:

- Clean and structured architecture
- Optimized and scalable system design
- Readable and maintainable code
- Security considerations
- Deployment and monitoring practices
- Automation where applicable
